# PlanningOps Domain Docs

planning 문서와 GitHub tracking 동기화의 cross-cut 정책을 이 경로에 축적한다.

## Activation Criteria
- cross-cut 정책 문서가 최소 1개 이상 생성되어야 한다.
- 문서 owner(리뷰 책임자)가 명시되어야 한다.
- 관련 링크가 `uap-docs.sh check`에서 통과해야 한다.
