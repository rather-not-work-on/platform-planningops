# Platform Contracts Docs

공용 계약(`C1~C*`) 관련 문서를 이 경로 하위에 배치한다.

## Activation Criteria
- 아키텍처 또는 실행계획 문서가 최소 1개 이상 생성되어야 한다.
- 문서 owner(리뷰 책임자)가 명시되어야 한다.
- 관련 링크가 `uap-docs.sh check`에서 통과해야 한다.
