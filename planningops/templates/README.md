# planningops/templates

## Purpose
Store canonical templates used to generate loop artifacts and contract evidence documents.

## Contents
- `ecp-template.md`
- `simulation-report-template.md`
- `handoff-packet-template.md`
- `delta-note-template.md`
- `gate-evidence-manifest.example.json`

## Change Rules
- Template field names must remain stable unless related contracts are updated first.
- Template examples should use repo-relative paths and ASCII-safe placeholders.
- Scripted generators must be updated together with template key changes.
